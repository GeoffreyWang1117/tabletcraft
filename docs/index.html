How It Works

Input English text
→
Classify Confidence gate
→
Translate ByT5 NMT
→
Validate Quality check
→
Render Clay tablet
Reliability first: The system classifies every input, validates every output, and degrades gracefully when uncertain. A tool that says "I'm not sure" is better than one that confidently renders wrong cuneiform.

Features

📜

14,240 Signs

Comprehensive transliteration-to-cuneiform mapping with 95.3% token coverage.

🔀

Bidirectional

Akkadian → English and English → Akkadian. Read ancient texts or write your own.

🛡

Confidence Gating

Input classification + output validation. Knows when it's unreliable.

🎨

Tablet Renderer

SVG/PNG clay tablets with authentic Mesopotamian styling. No GPU, <10ms.

⌨

CLI + API

pip install cuneiscribe. CLI, Python API, Gradio web demo.

📦

Batch Mode

Process CSV/JSONL files with full logging and failure reports.
